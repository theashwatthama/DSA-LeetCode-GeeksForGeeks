A personal, actively-growing collection of my **Data Structures & Algorithms** solutions from **LeetCode** and **GeeksForGeeks**, written in **Java**. Every solution folder is auto-annotated with runtime, memory usage, and percentile ranking via **LeetHub**, so you can see exactly how each solution performed.

---

## 📌 About This Repository

- ✅ Solutions organized by **problem number** and **topic/pattern**
- ✅ Each folder includes the working solution + LeetHub-generated performance stats (Time & Space complexity percentile)
- ✅ Tagged by **Difficulty** (Easy / Medium / Hard) for quick reference
- ✅ Focused heavily on core interview patterns: **Two Pointers**, **Sliding Window**, and **Slow & Fast Pointers (Floyd's Cycle Detection)**
